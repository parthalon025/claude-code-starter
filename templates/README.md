# CLAUDE.md Templates

These are ready-to-use project instruction files for Claude Code. Pick the one closest to your project, copy it, and customize.

## How to Use

1. Pick the template that matches your project type
2. Copy it into your project's root folder
3. Rename it to `CLAUDE.md`
4. Replace everything in `[brackets]` with your actual project details

**Or let Claude do it:** Open Claude Code in your project and say:
```
create a CLAUDE.md for this project based on what you see in the codebase
```

## Available Templates

| Template | Best For |
|---|---|
| `CLAUDE-general.md` | Any project (blank starting point) |
| `CLAUDE-react.md` | React + TypeScript + Tailwind projects |
| `CLAUDE-python.md` | Python + Flask/FastAPI + SQLAlchemy projects |
| `CLAUDE-nextjs.md` | Next.js 14 + Prisma + Tailwind projects |

## What is CLAUDE.md?

It is a file that tells Claude Code how to work in your project -- what tech stack you use, what commands to run, what rules to follow. Without it, Claude guesses. With it, Claude knows.
